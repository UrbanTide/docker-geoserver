# docker-geoserver

A simple docker container that runs Geoserver influenced by this docker
recipe: https://github.com/eliotjordan/docker-geoserver/blob/master/Dockerfile

# Steps
- clone this
- run `sh build.sh`
- run `sh run.sh`
